# Data

This is a Mode C project and does not have a static Mode B data pipeline.

Runtime case artifact contract:

- `reconstruction.json`
- `report.md`
- `case-bundle.zip`

The JSON schema is represented by the OpenAPI `Artifact` schema:

https://github.com/baditaflorin/accident-reconstructor/blob/main/api/openapi.yaml

`make data` intentionally prints an explanatory no-op message.
